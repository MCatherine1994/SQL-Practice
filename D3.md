#### **Selection**:  
##### Select by Name:  
```
d3.select('p')  // select the first <p> element
d3.selectAll('p')  // select all <p> element
```
##### Select by Id:  
```
d3.select('#elementId')
```
##### Select by ClassName:  
```
d3.select('.className')
```
